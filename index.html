What is this?

A Headless E-commerce Backend

WrightCommerce is not a website — it's the engine behind one. It exposes a RESTful API that powers storefronts, admin dashboards, and mobile apps.

🏪

Built for Small Businesses

Designed specifically for East African SMEs — local payments, local currencies, local needs.

🔌

Headless Architecture

Any frontend can connect — React, Vue, mobile apps, or even WhatsApp bots.

🔒

Secure & Token-Based

Bearer token authentication for all protected routes. Business and customer auth separated.

⚠️ This is an API-only service

This URL serves as the backend API for WrightCommerce-powered stores. There is no user interface here. If you are a customer looking to shop, please visit the storefront you were directed to.

Base URL: /public/api/v1/
Features

Everything a Store Needs

WrightCommerce Core ships with all the essentials built in — no plugins required.

📦

Product & Inventory Management

Full CRUD for products, categories, stock tracking, and image uploads.

🛒

Order Management

Order creation, status tracking, order items, and customer assignment.

💳

Paystack Payments

Card and M-PESA payments via Paystack with webhook verification.

👤

Customer Accounts

Customer registration, login, profile management, and order history.

🔔

Notifications

Email, SMS, and WhatsApp order notifications for customers and businesses.

🏢

Multi-Business Ready

Architecture supports multiple businesses on a single installation.
